# FPShowmany-idl

## Тест: Насколько хорош ваш телефон?

Приложение для стресс-тестирования производительности устройства Android.

### Функции

- **Тест CPU**: Нагрузка на процессор с использованием математических операций
- **Тест RAM**: Проверка производительности памяти
- **Тест GPU**: Стресс-тест видеокарты в режимах 2D, 3D или Авто
- **Система оценки**: 6 уровней производительности на русском языке

### Уровни производительности

1. **Очень Плохо** (0-1000 баллов)
2. **Плохо** (1001-3000 баллов)
3. **Неплохо** (3001-5000 баллов)
4. **Нормально** (5001-7000 баллов)
5. **Хорошо** (7001-9000 баллов)
6. **Отлично** (9000+ баллов)

### Сборка

```bash
# Сборка debug версии
./gradlew assembleDebug

# Запуск тестов
./gradlew test

# Очистка
./gradlew clean
```

### Требования

- Android Studio Hedgehog или выше
- JDK 17
- Android SDK 34

### GitHub Actions

Проект настроен с автоматической сборкой при пуше в main ветку.
